# 🕊️ Pastoral App - Sistema Full-Stack

Aplicación integral para la gestión pastoral, desarrollada con un stack moderno y escalable.

## 🚀 Tecnologías Utilizadas
* **Frontend:** React + Vite
* **Backend:** Node.js + Express
* **Base de Datos:** MongoDB Atlas (Mongoose)
* **Despliegue:** Render / Netlify

## 🛠️ Configuración Local
1. Clona el repositorio: `git clone https://github.com/mmotal79/pastoral-app.git`
2. Instala dependencias: `npm install`
3. Crea un archivo `.env` con tu `MONGODB_URI`.
4. Corre el proyecto: `npm run dev`

## 🌐 Despliegue en Render
Este proyecto está configurado para despliegue automático. 
**Variables requeridas:**
* `MONGODB_URI`: Cadena de conexión de Atlas.
